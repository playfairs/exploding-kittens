# Exploding Kittens

A simple game where you try to defuse as many kittens as possible before they explode.

>[!NOTE]
> This game is not at all ready to be played, it's being worked on slowly, and I need to make the sprites, sound effects, and more, so this will take a while.

## Controls

- Left Click: Defuse a kitten

## How to Play

1. Select a kitten skin
2. Click to defuse kittens before they explode
3. Try to defuse as many kittens as possible
4. The game is over when 3 kittens explode
